# Fake News Detection

A machine learning model that classifies news articles as Real or Fake 
using NLP techniques including TF-IDF vectorization and multiple classifiers.

## Tech Stack
Python, Scikit-learn, Pandas, NumPy, TF-IDF, Regex

## Models & Accuracy
- Logistic Regression: 98.81%
- Decision Tree: 99.51%
- Random Forest: 99.80%
- Gradient Boosting: 99.49%

## How to Run
pip install pandas numpy scikit-learn
python fakenews.py

## Dataset
Fake and Real news dataset included in the repository.
